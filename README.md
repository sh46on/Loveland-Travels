# 🌍 Loveland Travels

A modern travel and tourism web application built using **React + Django**.
Loveland Travels helps users explore destinations, view packages, read reviews, and make enquiries seamlessly.

---

## ✨ Features

* 🏝️ Explore popular destinations
* 🧭 Tour packages with detailed information
* ⭐ User reviews and ratings
* 📩 Enquiry / contact system (EmailJS integration)
* 📱 Fully responsive design (mobile + tablet friendly)
* ⚡ Fast and modern UI with React

---

## 🛠️ Tech Stack

### Frontend

* React.js
* CSS3 (Custom styling)
* EmailJS (for contact forms)

### Backend

* Django
* Django REST Framework

### Database

* MySQL

### Hosting

* Frontend: Netlify
* Backend: PythonAnywhere

## 🚀 Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/sh46on/Loveland-Travels.git
cd Loveland-Travels
```

---

### 2️⃣ Frontend Setup

```
cd frontend
npm install
npm start
```

---

### 3️⃣ Backend Setup

```
cd backend
pip install -r requirements.txt
python manage.py runserver
```

---

## 🔐 Environment Variables

Create a `.env` file in frontend for EmailJS:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
